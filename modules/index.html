Modules

The source code is in a folder named src. The top level application is in main.py. A directory named tidegates has modules for different sections of the GUI.

src
├── main.py
└── tidegates
    ├── budgets.py
    ├── messages.py
    ├── optipass.py
    ├── project.py
    ├── styles.py
    ├── targets.py
    └── widgets.py
